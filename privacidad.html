FINALIDADES DE TRATAMIENTO

Los datos que se dan a cabo por el usuario serán tratados con las siguientes finalidad según su tipo de formulario:

A) FORMULARIO DE CONTACTO

  • Disponerle todo tipo de información al usuario sobre la solicitud realizada.

B) FORMULARIO DE REGISTRO DE USUARIO

  • Dar acceso a la creación de usuario
  • Enviar comunicados en relación con la cuenta creada

C) FORMULARIO DE COMPRA / PEDIDO

  • Realizar la facturación correcta al usuario
  • Cumplir con las obligaciones legales

D) FORMULARIO DE SUSCRIPCIÓN A NEWSLETTER

  • Gestionar la suscripción y asegurar el envío de correos y folletos publicitarios.
  • Visualizar el comportamiento y el interés del usuario a la hora del envío.

3. BASE JURÍDICA

  • Tener consentimiento a la hora de enviar suscripciones a newsletter
  • Contratos seguros a la hora de realizar una compra
  • Interés por parte del responsable para llevar a cabo un mejor rendimiento de la web.

4. DATOS RECOGIDOS

El encargado / responsable deberá de obtener todo tipo de datos personales como:

  1. Nombre, apellidos y DNI
  2. Dirección postal, correo electrónico y teléfono
  3. Acceso a toda la información a la hora de comprar
  4. Tipo de navegador, páginas más visitadas y el tiempo que transcurre en ellas.

5. PLAZO DE CONSERVACIÓN DE LOS DATOS

Los datos personales se conservarán solamente durante el tiempo necesario para cumplir con las finalidades marcada, en especial:

  • El tiempo para atender la solicitud
  • El número de usuario que entrena en la web
  • El envío de newsletters hasta que el usuario se dé de baja

6. DESTINATARIOS

Los principales destinatarios serán las empresas de transporte que facilitan la entrega de envíos a nuestros clientes, entidades bancarias para el proceso de compras y administraciones públicas para las facturas.

7. DERECHOS DE LOS USUARIOS

El usuario tiene derecho a los siguientes derechos:

  • Retirar el consentimiento de las newsletter
  • Solicitar la eliminación de sus datos
  • Modificar datos erróneos
  • Saber qué tipo de datos queremos y necesitamos obtener
  • Recibir datos sobre los artículos ecológicos

8. CÓMO EJERCER LOS DERECHOS

Los derechos del usuario se verán implementados de manera escrita acompañada de una copia de los datos de identidad a través de correo electrónico (spinley_ecologic@gmail.com) y correo postal (Av. Corazón de Jesús, 12600 La Vall D'uixó, España).
